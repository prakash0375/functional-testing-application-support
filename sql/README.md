# Functional Testing & Application Support Project

## Overview
This repository demonstrates Functional Test Planning, Test Case Design, Defect Tracking, Root Cause Analysis, and Application Support simulation for a User Authentication System.

## Key Skills Demonstrated
- Functional Test Planning
- Test Case Design & Execution
- Regression Testing
- Defect Tracking
- Root Cause Analysis
- Requirement Traceability Matrix (RTM)
- SQL Data Validation
- Troubleshooting & Issue Resolution

## Project Structure
- docs/ → Testing documents
- sql/ → Validation queries

This project simulates real-world application support and functional testing processes.
